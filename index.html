This homepage is now the entry point for a personal knowledge graph. It separates durable knowledge assets from practice projects, so humans can browse concepts clearly and agents can retrieve structured facts, links, examples, and verification steps with less guesswork.

What lives here now

The homepage no longer lists all practice projects directly. Instead, it points to the durable knowledge layer: a graph schema, a graph snapshot, and node documents written for both human study and future agent retrieval pipelines.

Why this structure works

  • Humans can read node Markdown like study notes.
  • Agents can traverse JSON nodes and edges deterministically.
  • Projects are preserved as evidence, not mixed into the homepage information architecture.
  • Future scripts can export to JSONL, embeddings, or graph visualization libraries.
