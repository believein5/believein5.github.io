# Sources

This folder stores **source nodes** for provenance in Scheme B.

## Purpose

- Keep references (books, papers, sections) as independent entities.
- Let knowledge nodes link to sources via:
  - `defined_in`
  - `supported_by`
  - `cited_from`

## Naming

Use kebab-case file names that match node IDs when possible.

Example:

- node id: `src-aima-4e-ch1-sec1-1-4`
- file: `aima-4e-ch1-sec1.1.4.md`
