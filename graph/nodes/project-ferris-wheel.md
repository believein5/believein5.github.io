## Ferris Wheel

## Role in the graph

This project validates that animation concepts have been translated into a working artifact.

## Key concepts demonstrated

- Continuous rotation with `@keyframes`
- Independent cabin animation
- Absolute positioning for repeated parts
- Visual timing and motion consistency
