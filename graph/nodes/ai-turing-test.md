## Definition

A system passes the Turing test if a human interrogator cannot reliably distinguish it from a human by text responses.

## Required capabilities

- natural language processing
- knowledge representation
- automated reasoning
- machine learning

Total Turing test additionally needs perception and action (e.g., vision and robotics).
